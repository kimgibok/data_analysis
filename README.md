## 러닝타임별
- 러닝타임별 막대그래프
if 내가 공연을 처음보러가는사람:
 내가 보는 공연이 너무 긴거 같은데 평균적으로 러닝타임이 어떻게 되나? 오 나는 긴편은 아니구나
